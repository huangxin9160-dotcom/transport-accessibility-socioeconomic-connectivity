# Data dictionary

## si_ni_ccd_results.csv

| Variable | Description |
|---|---|
| agglomeration_id | Abbreviation of the urban agglomeration |
| agglomeration_name | Full English name of the urban agglomeration |
| Si or S_i | Spatial Integration index |
| Ni or N_i | Social Networks Connectedness index |
| CCD | Coupling Coordination Degree |
| coupling_regime | High-Coupling, Social-Lagging, Spatial-Lagging or Low-Coupling |

## cr5_2000_2020_results.csv

| Variable | Description |
|---|---|
| CR5_2000 | Share of total socioeconomic flow captured by the five strongest nodes in 2000 |
| CR5_2020 | Share of total socioeconomic flow captured by the five strongest nodes in 2020 |
| delta_CR5 | CR5_2020 minus CR5_2000 |

## service_zone_area_statistics.csv

| Variable | Description |
|---|---|
| full_service_percent | Share of area accessible within all three time thresholds |
| dual_service_percent | Share of area accessible within two thresholds |
| single_service_percent | Share of area accessible within one threshold |

## figure_source_data

The files in `figure_source_data/` provide figure-level source data for Figs. 1-6. Replace template rows and blank values with the final public values used in the submitted manuscript before archiving the repository.
