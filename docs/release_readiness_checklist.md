# Release readiness checklist

Use this checklist before creating a GitHub release and Zenodo DOI:

- [ ] Replace `TO_BE_UPDATED` in `LICENSE` with the copyright holder.
- [ ] Replace author placeholders in `CITATION.cff`.
- [ ] Replace `USERNAME` in `CITATION.cff` after the GitHub repository URL is known.
- [ ] Confirm the GitHub repository is public.
- [ ] Confirm `derived_data/` contains only shareable aggregate results and no restricted raw data.
- [ ] Replace all blank/template values in `derived_data/` with final manuscript values.
- [ ] Replace all blank/template values in `figure_source_data/` with final public figure source data for Figs. 1-6.
- [ ] Confirm `data_sample/` contains only toy/sample records.
- [ ] Confirm `docs/restricted_data_notice.md` describes raw data restrictions and access conditions.
- [ ] Create a GitHub release.
- [ ] Enable the repository in Zenodo.
- [ ] Confirm Zenodo metadata and copy the DOI URL.
